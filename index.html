Asterisk-Flite by zaf

Asterisk module that provides the "Flite" dialplan application, which allows you to use the Flite TTS Engine with Asterisk.

Flite For Asterisk 1.6 This provides the "Flite" dialplan application, which allows you to use the Flite TTS Engine with Asterisk. This module invokes the Flite TTS engine locally, and uses it to render text to speech.

Dependencies

Asterisk development header files
Flite libraries and header files

Install

To build just type the following in the app-Flite folder
$ make
$ make install

To install the sample configuration file, issue the following command after the 'make install' command:
$ make samples
